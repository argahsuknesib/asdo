Aggregating Stream Data Ontology

Current Version: https://argahsuknesib.github.io/asdo/

Ontology in development

Abstract

Aggregating Stream Data is important for streams generated from distributed data sources.

Introduction

In a decentralised environment, aggregation data neccesary for summarising the generated data streams. There is a need for an ontology to describe the services which are built upon the data sources or the sensors the stream is generated from. In this ontology, we extend the Stream Annotation Ontology to include the services built on top to do aggregation of streams.
